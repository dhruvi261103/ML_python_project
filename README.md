 Project Objectives
Data Preprocessing:
Handle missing values and scale numerical features.
Normalize the dataset using StandardScaler.

Model Development:
Apply One-Class SVM for anomaly detection.
Apply Local Outlier Factor (LOF) to identify outliers.
Compare the results and mark anomalies.

Evaluation and Result Analysis:
Analyze detected anomalies using Glucose and BMI as key features.
Visualize anomalies for better interpretation.
Save the results to diabetes_anomaly_results.csv.

Visualization with Power BI:
Develop an interactive Power BI dashboard (ML DASHBOARD.pbix).
Highlight key insights, anomaly trends, and patient analysis.
